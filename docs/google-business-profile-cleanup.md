## Per-profile procedure

1. Search Google Maps for the exact parish name and city.
2. Open the existing result and choose **Own this business?** or request manager access when necessary.
3. Confirm the existing owner before starting a new verification.
4. Compare name, address, phone, map pin, category, website, hours, accessibility attributes, and photos with the approved values above.
5. Correct one field at a time and save evidence/screenshots of significant changes.
6. Add the specific formal parish-page URL only after it works publicly.
7. Review user-supplied photos and Q&A for inaccurate claims, but do not argue with reviews or create fake reviews.
8. Check the public result while signed out after Google approves the changes.
